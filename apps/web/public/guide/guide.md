# 周末好去处

1. 杭电周边：西门 328 路到终点站，走两步转地铁1号线（7元，坐车不到1小时大概）；

    1. 至金沙湖站，可得1商场

    2. 至高沙路站可得2商场\+高沙商业街开吃

    3. 至文泽路可得2商场

    4. 至文海南路站可得1商业街\+宝龙广场；

    5. 杭电食堂：走学林街的那俩门就行，或者闸机尾随或刷身份证件；食堂付款问阿姨叔叔要后勤的码或者拉个人帮忙付一下，不然多出20%；

2. 4公里处的中国水利博物馆、水博园，上塔20元。

3. 西湖美景：以下杭州推荐来自我的挚友，感谢

    1. 经典线路:断桥\-\-\-白堤\-\-\-平湖秋月\-\-孤山公园\-\-浙江省博物馆\(可选\)\(可结合你写的坐船的线路\)

    2. 苏堤线路:可做地铁到黄龙洞下，走宝石山，下山就是岳王庙，如果不想走太多可以直接到岳王亩，然后走苏堤，欣赏西湖另一面的景色，苏堤走到头的左手边应该就是雷峰塔了，如果第一次去可以进去看看，是花门票的，当然我个人觉得不是那么值得进塔里，毕竟里面只有地基是遗址了。\(可以天色晚的时候走这条线路\)

    3. 西湖其他周边景点根据你们自己时间可选，杨堤、曲院风荷\(夏天赏荷花，四五月份应该还没有\)、花港观鱼\(看大鲤鱼\)、钱王祠、岳王庙

4. 西溪湿地公园

    1. 里面蛮大的，适合赏花、露营，可以玩一小天，有很多条线路，我们当时走的是主干的线路，小红书上一搜就有。公园免费的，里面也有饭店，不过肯定较贵。如果想尽可能游完西湖，西溪湿地可以不选，毕竟也是要徒步一天，很消耗体力。里面最有特色的就是各种花，各种树，各种池塘。

5. 古街

    1. 若是当天白天游玩时间结束很早，傍晚可以去一些很有特色的江南街道走走，例如南宋御街、河坊街、小河直街。一定不要在中午或者下午两三点去，可能会很热。这三个我都去过，南宋御街应该是最大的街区了，有特色的门店也可以去逛逛，河坊街也类似，不过小河直街因为中间有个小河分割，所以很有江南水乡的韵味，如果想拍花的服饰照片，应该只有这里有，价格要货比三家，装造什么都很不一样。

6. 京杭大运河

    1. 很有历史文化的地方，可以乘坐三块的水上巴士进行游览。我当时是在武林广场站上的船，拱宸桥下的船，每趟船都有固定时间，好像是间隔15\-30分钟一趟。到拱宸桥那有很多值得游览的地方，可以小红书上看看，如果没记错的话，小河直街应该也不远。

7. 龙井村或者龙坞茶庄

    1. 都是以龙井茶而闻名，最出片的就是一大片一大片的茶山，一般四月份清明前后是最壮观的，当然要想买茶叶，龙井的明前茶也是不错的选择，但是一定要选择正确的渠道，里面家家基本都是种茶的，可以进家中购买，不要在村口买，很可能是假的，或者质量不好的。

    2. 龙井村肯定要比龙坞有名，但其实差不了多少，节假日的时候龙并村肯定人满为患，所以不如去龙坞茶庄，路线小红书也有，比如什么光明寺水库，足够玩一天了。

8. 灵隐寺

    1. 灵隐寺是位于飞来峰景区里面，所以要想去灵隐寺，先得买进飞来峰景区的门票。当然，灵隐寺也是收费的，要是不去灵隐寺，在飞来峰景区里游玩也是可以的。飞来峰不高，灵隐寺就在山脚，景区比较出名的就是各种石佛，其他的就是花、别的一些寺庙等等

    2. 除了茶庄和灵隐寺，其他地方都有地铁直达，这两个地方当然也有专线的大巴。杭州市区我去过的有名的景区大概也就这些了，主要还是围绕西湖展开游玩，具体的地方小红书上也有攻略。

9. 闪游上海、南京、南昌、苏州、武汉、合肥

# 奇门技巧

1. 工位椅子可以调整靠背的反弹力度，拧座椅下面的圆柱形装置即可，看准标注的 loose和 tight 方向，最好找肌肉男，不然容易伤到 O\(∩\_∩\)O。不想靠下去的时候就把右下边“扳手”插进去，拔出来就可以靠下去了。

2. 园区内中央空调面板一般都会常亮，开关按钮亮起才表示空调开启；特别的是A8楼非上班时间中央空调停摆，开热风会吹冷风。

3. 宿舍空调长按最左和最右按键三秒进入工程模式，选到C05切换成1可以开启室内温度显示，C07可以修改室内温度的偏移量，空调将会按照室内温度和设定温度进行对比来进行制冷和制热，如果你的空调一直制冷不停下可以尝试修改这个设置。

    1. 如果你突然发现空调不工作了（重启、等待都没用），可以试试关掉空调的电闸再打开，一般就可以恢复了。

    2. 如果你发现面板上显示奇怪的报错，也可以关闭空调，等待会再打开看有没有报错，如果报错一直始终存在，则可以找宿管保修了。

4. 校园卡去食堂刷一下激活，然后就可以在宿舍插卡用热水，但插卡用的热水不会有限额，忘记拔卡或者暂停会损失重大，建议用手机一卡通服务中的用水服务开启，设置限额或者设置限时。

5. 宿舍水电费是一个账户，空调比较费电，冬天一个小时快1块钱，夏天还没过不知道。收费标准：冷水 2\.95 元/吨（西电学生每人每月免费3吨）；热水 0\.27 元/分钟；电费 0\.538 元/度（西电硕士生每人每月免费10度，西电博士生每人每月免费12度）直接补贴到水电费余额。

6. 宿舍卫生间弱暖似乎吹的是冷风，冬天建议强暖，似乎强暖有点太热了……。

7. 体育馆302有音乐教室，架子鼓、钢琴、吉他等设备齐全……轻音乐社群197698007管辖，欢迎加入音乐的世界\~

# 杭研园区内关于双电脑使用场景

远程方案需要依靠高质量的局域网、外网传输速度：

**一般版**

方案1：配套1080P显示器连接配套电脑，笔记本放旁边，一套键鼠通过工具连接。

方案2：工位主机、宿舍笔记本，互不干扰，有需要远程。

**体验版**

购置一4K/2K高刷显示器，双电脑同时连接至一台显示器，需要时切换或远程，一套键鼠通过工具连接。

**土豪版**

购置双4K/2K高刷显示器，一横一竖随意摆放，分别或一起连到电脑，一套键鼠通过工具连接。

**一些操作**

● 键鼠共享工具参考：[https://sspai\.com/post/72143](https://sspai.com/post/72143)

● VSCode Remote\+SSH 开发，一台当主力机，另一台当开发机。

● 不论系统开启 SSH 远程连接或者设置共享文件夹以快速同步文件（注意SSH安全）。

● 有线无线不在同一网段无法嗅探到，但有线无线是在同一个大局域网内可通信的。

● 微软自带 RDP（远程桌面连接，需要Win专业版）远程连接体验很好，在杭研里局域网都可连接，且自动同步键鼠、可拖文件，推荐使用，外网可备用 ToDesk、向日葵等等，缺点是只有30FPS，改注册表提高帧数方法不太灵光。

● 有打游戏需求可采用 Sunshine \+ MoonLight 串流，保障有线传输，可支撑高分辨率、高刷新率及低延迟的效果。当然也可以用来远程，实测 N 卡开启YUV444编解码效果不错，A卡有点模糊不如RDP但也能用，集显也可以的。

● RustDesk 比上面的串流方案多了远程协助的很多附带功能，也很推荐。

● AnyDesk 是唯一局域网内非N卡有YUV444色彩表现且原生60FPS的，也可以试试，缺点就是有些窗口点不动。

● 局域网内文件传输可以使用 LocalSend，Github项目。

# 无法连接有线网络问题解决方案

~~杭研无线网络限速50M，有线网络为1000M上下行对等。~~

连接有线网络后，弹出认证网页但始终无法完成认证，如图所示：

![filename\.png](Images_attachments/filename6.png)

未知，猜测深澜计费系统对某些电脑的MAC地址存在格式校验漏洞，导致部分电脑的MAC地址无法被系统核验通过加入认证。

经与信息中心沟通，学生需要在认证系统的无感知认证中加入自己电脑的MAC地址后，即可自动联网，具体操作步骤如下：

1、系统设置、网络、高级网络设置

![filename\.png](Images_attachments/filename2.png)

2、查看物理地址信息，复制一份

![filename\.png](Images_attachments/filename4.png)

3、使用WIFI连接深蓝认证系统，电脑和手机均可操作

![filename\.png](Images_attachments/filename1.png)

4、点击自助服务、左侧用户、无感知认证，右侧添加MAC

![filename\.png](Images_attachments/filename5.png)

5、将复制的MAC地址粘贴，横杠改冒号，大写改小写！否则校验失败！

例如，00\-E5\-4B\-76\-C6\-2D 需要改成 00:e5:4b:76:c6:2d

![filename\.png](Images_attachments/filename7.png)

6、增加按钮，看到列表中新增一项，成功。

7、重新连接有线网络，自动连接成功。
